# 作業記録: 参照関係の統一

**作成日時**: 2026-02-07 17:11:17  
**タスク**: Issue 5 - 参照関係の統一（中優先）

---

## タスク概要

### 目的
すべてのsteeringファイルの「関連ドキュメント」セクションを統一形式に変更する。

### 背景
現在、steeringファイルの「関連ドキュメント」セクションには以下の問題がある：
- 形式が統一されていない（「参照元」「参照先」の区別がある）
- 複雑な参照関係の表現が保守性を低下させている
- シンプルな相互参照形式に統一することで、可読性と保守性を向上させる

### 目標
1. すべてのsteeringファイルの「関連ドキュメント」セクションを確認
2. 統一形式に変更：`- **ファイル名**: \`相対パス\` - 簡潔な説明`
3. 「参照元」「参照先」の区別を削除
4. すべてのファイルで一貫性を確保
